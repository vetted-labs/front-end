Implementation Summary

High Impact
  • 1. Homepage Hero — sets the brand tone instantly
  • 2. Guilds Listing Hero — the most "brand" page
  • 3. Guild Cards — gives every card a unique texture
  • 4. Browse Hero — main public discovery surface
Supporting
  • 5. Stats Bar — very subtle, adds texture depth
  • 6. Footer — bookends the page nicely
  • 7. Leaderboard Hero — "stage" atmosphere
  • 8. Guild Detail Header — banner identity
Technique
  • • Use the actual SVG pattern files (BackGround.svg) as background-image just like login page does
  • • Control with CSS mask-image gradients for fade-in/out edges
  • • Keep opacity at 3-6% — pattern should be felt, not read
  • • Light mode: use pattern-light.svg / Dark mode: use pattern-dark.svg (already have both)
  • • Create a shared <PatternBackground /> component for reuse
